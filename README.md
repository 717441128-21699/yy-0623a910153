# 正畸附件记录本

面向小型个人正畸诊所医生的极简桌面端应用，用于记录附件粘接情况。

## 功能特点

- **患者档案**：管理患者基本信息（姓名、病历号、矫治器品牌、当前副数）
- **快速记录**：牙位图点选粘接牙位、选择附件形态、填写备注、拖放照片，系统按日期形成时间线
- **复诊对比**：并排查看两次记录，标记每颗牙的状态（完好、磨耗、脱落、重粘）
- **离线使用**：所有数据存储在本地 SQLite 数据库，无需网络
- **极简设计**：上手快，操作清晰，适合一两台椅位的小诊所

## 技术栈

- Electron - 桌面端框架
- Vue 3 - 前端框架
- Vite - 构建工具
- better-sqlite3 - 本地数据库

## 运行方式

```bash
# 安装依赖
npm install

# 开发模式运行
npm run electron:dev

# 构建生产版本
npm run electron:build
```

## 数据存储

- 数据库文件：系统用户数据目录下的 `ortho.db`
- 照片文件：系统用户数据目录下的 `photos/` 文件夹
